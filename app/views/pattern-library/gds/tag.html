Tag

Use the tag component to show users the status of something — for example whether an application is submitted, in progress or approved.

Default tag colours

The available colour modifiers are designed to convey meaning. Use them consistently so users can build familiarity across the service.

Tags in tables

Tags are most commonly used in case lists to show status at a glance. Here is how they appear in a table context.

Example
Application Applicant Status
100300 Berkshire Estate Limited In review
100297 Watson Farms Ltd Confirmed
100284 Greenfield Agricultural On hold

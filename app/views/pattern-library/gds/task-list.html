Task list

Use the task list pattern to give users a clear overview of the steps they need to complete in a long or complex transaction.

Default task list

Each task has a name, optional hint, and a status tag indicating progress.

Example
  • Check customer details
    In progress
  • Review land parcel rule calculations
    Ready to start
  • Confirm payment amount
    Cannot start yet
  • Review scheme budget as a finance officer
    Completed

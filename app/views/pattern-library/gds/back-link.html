Back link

Use the back link component to help users go back to the previous page in a multi-page transaction.

In this prototype the back link appears in the green context strip below the Defra header, allowing caseworkers to return to the case list from any case detail page.

Back 
<a href="/caselist" class="govuk-back-link">Back</a>

Back link in context strip

In this prototype the back link sits inside the context strip alongside case metadata. The GOV.UK guidance recommends placing it at the top of the page, which this pattern satisfies.

Warning text

Use the warning text component when you need to warn users about something important, such as legal consequences of an action.

Use this for high-stakes information that the caseworker must not overlook — for example confirming that a rejection decision is irreversible.

Default warning text

Example
! Warning Rejecting this application cannot be undone. The applicant will be notified immediately.

  <span class="govuk-warning-text__icon" aria-hidden="true">!</span>

    <span class="govuk-visually-hidden">Warning</span>
    Rejecting this application cannot be undone.
    The applicant will be notified immediately.
  </strong>
</div>

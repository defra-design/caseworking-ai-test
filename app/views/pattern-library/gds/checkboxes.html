Checkboxes

Let users select one or more items from a list.

Use checkboxes when a question can have any combination of answers (zero, one, or several). If only one answer is possible, use radios instead.

Small checkboxes

Use the govuk-checkboxes--small modifier inside a dense filter panel where the standard control would dominate the layout. Used in the My case list filters.

When to use

  • The question accepts zero, one, or more answers from a known list.
  • Inside a filter panel, where the user might combine criteria (status A and status B and assignee X).

When not to use

  • The question is mutually exclusive — use radios.
  • The list is long and unbounded — use the accessible autocomplete for a single value, or paginated search.

Where does this occur

  • My case list filters — small checkboxes for scheme type, status and assignee filters.

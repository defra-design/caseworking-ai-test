Button

Use the button component to help users carry out an action like starting an application, submitting a form or saving progress.

Default button

Use a default button for the main call to action on a page. Avoid using multiple default buttons on one page.

Example 
<button type="submit" class="govuk-button" data-module="govuk-button">
  Save and continue
</button>

Secondary button

Use secondary buttons for secondary calls to action on a page. Pages should only have one primary action.

Example 
<button type="button" class="govuk-button govuk-button--secondary" data-module="govuk-button">
  Find address
</button>

Warning button

Warning buttons are for destructive actions like permanently deleting data. Give careful consideration to when to use these.

Example 
<button type="button" class="govuk-button govuk-button--warning" data-module="govuk-button">
  Delete account
</button>

Button group

Use button groups to display a primary action alongside a secondary action or link.

Example
Cancel 
<div class="govuk-button-group">
  <button type="submit" class="govuk-button" data-module="govuk-button">
    Save and continue
  </button>
  <button type="button" class="govuk-button govuk-button--secondary" data-module="govuk-button">
    Save as draft
  </button>
  <a class="govuk-link" href="#">Cancel</a>
</div>

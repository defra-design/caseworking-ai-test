Summary list

Use the summary list to summarise information, for example a user's responses at the end of a form.

Default summary list

The most common usage in this prototype is displaying application details on case overview pages.

Example
Name
Sarah Watson
Business name
Berkshire Estate Limited
SBI number
112849012
Contact details
sarahfarmer@gmail.com
07700 900457
<dl class="govuk-summary-list">
  <div class="govuk-summary-list__row">
    <dt class="govuk-summary-list__key">Name</dt>
    <dd class="govuk-summary-list__value">Sarah Watson</dd>
  </div>
  <div class="govuk-summary-list__row">
    <dt class="govuk-summary-list__key">Business name</dt>
    <dd class="govuk-summary-list__value">Berkshire Estate Limited</dd>
  </div>
</dl>

Summary list with actions

Include a change link alongside each row when users need to be able to amend their answers.

Example
Name
Sarah Watson
Change name
Date of birth
5 January 1978
Change date of birth
<dl class="govuk-summary-list">
  <div class="govuk-summary-list__row">
    <dt class="govuk-summary-list__key">Name</dt>
    <dd class="govuk-summary-list__value">Sarah Watson</dd>
    <dd class="govuk-summary-list__actions">
      <a class="govuk-link" href="#">
        Change <span class="govuk-visually-hidden">name</span>
      </a>
    </dd>
  </div>
</dl>

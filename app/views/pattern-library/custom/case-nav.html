Case navigation

A horizontal service navigation bar that lets caseworkers move between the main tabs on a case: Tasks, Application, Calculations, Timeline, Notes and Agreement.

Built on the GOV.UK govuk-service-navigation component with two custom item classes: .govuk-service-navigation__item-c (Tasks, wider margin) and .govuk-service-navigation__item-c2 (all other tabs, wider right margin at desktop). Do not change these classes globally — they affect all case pages.

The active tab is driven by the activeNavItem Nunjucks variable set before including the nav partial. Tabs are conditionally shown based on session state: Application is hidden in the 5-month stage; Agreement is only shown once the case reaches that stage.

Main case navigation

Used on the main case (application 100300). References data.caseStage and data.agreementStage.

Flex radio layout

A horizontal flexbox layout for radio groups and checkboxes used where vertical stacking is too space-inefficient for dense decision forms.

The GOV.UK Design System recommends vertical stacking for radios in most cases. This deviation is used selectively on task review screens where the decision label and radio options sit side by side to reduce vertical scroll on data-heavy pages.

! Warning Only use this pattern in contexts where it already appears. Use standard GOV.UK vertical stacking for any new radio groups unless the surrounding pattern already uses the flex layout.

radio-flex-box

Places a label and a radio group inline with a 1rem gap. Used on task review screens where a short action label precedes a small set of options.

Example 
/* application.scss */
.radio-flex-box {
  display: flex;
  align-items: center;
  gap: 1rem;
  margin-bottom: 1rem;
}

<!-- Usage -->
<div class="govuk-form-group">
  <div class="radio-flex-box">
    <label class="govuk-label govuk-!-font-weight-bold">Decision</label>
    <div class="govuk-radios govuk-radios--inline" data-module="govuk-radios">
      <div class="govuk-radios__item">
        <input class="govuk-radios__input" id="decision" name="decision"
               type="radio" value="Accepted">
        <label class="govuk-label govuk-radios__label" for="decision">
          Accepted
        </label>
      </div>
    </div>
  </div>
</div>

radio-flex-box2

.radio-flex-box2 is a variant with a fixed left margin (215px) and top margin (60px) used to align a secondary radio group against a specific label column width in the task review tables. Retain it where it already appears; do not introduce it into new layouts.

Example 
/* application.scss */
.radio-flex-box2 {
  display: flex;
  align-items: center;
  margin-left: 215px;
  margin-top: 60px;
  gap: 1rem;
  margin-bottom: 1rem;
}

<!-- Usage -->
<div class="radio-flex-box2">
  <div class="govuk-radios govuk-radios--inline" data-module="govuk-radios">
    <div class="govuk-radios__item">
      <input class="govuk-radios__input" id="decision" name="decision"
             type="radio" value="Confirm">
      <label class="govuk-label govuk-radios__label" for="decision">
        Confirm
      </label>
    </div>
  </div>
</div>

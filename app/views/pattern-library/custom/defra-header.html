Defra header

A fully custom internal header used on all casework pages in place of the standard GOV.UK or MOJ header.

The header uses the .defra-internal-header class with a 8px solid bottom border in Defra brand green (#00a33b). It includes the Defra logo on the left and the service name on the right.

Do not replace this with a standard GOV.UK or MOJ header. All changes to this component should remain within the .defra-* classes in application.scss.

Manage rural grant applications

        Manage rural grant applications
      </a>
    </div>
  </div>
</header>

Key classes

Class Purpose
.defra-internal-header Outer wrapper — sets padding and the 8px green bottom border
.defra-logo Logo container — green left border, sets min-width for alignment
.defra-logo__link Renders the Defra PNG logo via CSS ::before pseudo-element
.defra-internal-service-name Service name container — vertically centred alongside the logo
.defra-internal-service-name__link Service name link — 21px / 24px font, no underline, black text

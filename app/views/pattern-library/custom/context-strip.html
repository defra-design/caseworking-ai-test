Context strip

A light-green strip (.lg-div) below the Defra header that provides persistent case context on every case detail page.

The strip contains the back link, the applicant name as the page heading, and key case metadata (application ID and status tag). This keeps caseworkers oriented without needing to scroll back to the top of dense task lists.

Background colour: #E7F4EB. Do not change this to a GOV.UK colour token — the specific green tint is intentional branding.

Main case context strip

Used on all main-case casework pages. References data['caseStatus'] for the status tag.

Example
Back to case list

Berkshire Estate Limited

Application: 100300    In review 

<div class="lg-div govuk-!-padding-4">
  <div class="govuk-width-container">
    <a href="/caselist" class="govuk-back-link">Back to case list</a>
    <h1 class="govuk-heading-m govuk-!-margin-top-2 govuk-!-margin-bottom-1">
      Berkshire Estate Limited
    </h1>
    <p class="govuk-body-s govuk-!-margin-bottom-0">
      Application: <strong>100300</strong>
      <strong class="govuk-tag {{ data['caseStatusTag'] }}">
        {{ data['caseStatus'] }}
      </strong>
    </p>
  </div>
</div>

Linked case (case2) context strip

Spacing

Spacing follows the GOV.UK spacing scale. Use the GOV.UK spacing override classes in markup and govuk-spacing() in Sass. The $cw-space-* tokens mirror the scale for the rare case where a token is needed.

This replaces the magic-number classes that built up in the old CSS: .margin-left-177, .margin-top-200, .custom-width-130 and so on. See Layout & grid for how to replace them.

The scale

Example 
// _tokens.scss — mirrors govuk-spacing(0..9)
$cw-space-1: 5px;   $cw-space-2: 10px;  $cw-space-3: 15px;
$cw-space-4: 20px;  $cw-space-5: 25px;  $cw-space-6: 30px;
$cw-space-7: 40px;  $cw-space-8: 50px;  $cw-space-9: 60px;

How to apply spacing

Need Do this
Space below an element in markup class="govuk-!-margin-bottom-4" (the 1–9 suffix is the scale point)
Padding in markup class="govuk-!-padding-3"
Spacing in a Sass rule margin-bottom: govuk-spacing(4);
Gap between flex/grid children gap: $cw-space-4; (see .cw-row)

Do and don't

Do

  • Use scale points so vertical rhythm stays consistent.
  • Use the grid for column alignment.
  • Use gap on a flex/grid row, not per-child margins.

Don't

  • Add one-off classes like .margin-left-177.
  • Nudge layout with pixel margins to fake alignment.
  • Use .margin-top-200 to push content down the page.

Where does this occur

Everywhere. The migration away from magic-number spacing is tracked in MIGRATION.md.

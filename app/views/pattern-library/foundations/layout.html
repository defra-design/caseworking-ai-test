Layout & grid

Pages are laid out with the GOV.UK grid inside a govuk-width-container. Use the grid for columns and the semantic .cw-* classes for repeating layouts — never magic-number margins.

Split pane (library & case screens)

A persistent left navigation beside a content column. The pattern library itself uses .app-split-pane; case screens use the same idea with the case navigation.

Migrating magic numbers

Replace one-off alignment margins with the grid; replace fixed widths with a .cw-col-* class. The full table is in MIGRATION.md. Legacy class names keep working as shims until each screen is migrated.

Where does this occur

Caselists, case overview and calculations rely most on column widths and the split-pane layout.

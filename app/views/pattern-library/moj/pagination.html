Pagination

Use the MOJ pagination component to allow users to navigate between pages of results in a case list or search results view.

For service-style list pages that don't already use MOJ filter / sortable-table, prefer the GOV.UK Design System variant — see GDS pagination.

Default pagination

Example

Showing 1 to 25 of 143 results

  • Previous set of pages
  • 1
  • 2
  • 3
  • ⋯
  • 6
  • Next set of pages 
<nav class="moj-pagination" aria-label="results">
  <p class="moj-pagination__results">
    Showing <b>1</b> to <b>25</b> of <b>143</b> results
  </p>
  <ul class="moj-pagination__list">
    <li class="moj-pagination__item moj-pagination__item--prev">
      <a class="moj-pagination__link" href="#">Previous</a>
    </li>
    <li class="moj-pagination__item moj-pagination__item--active">
      <a class="moj-pagination__link" href="#">2</a>
    </li>
    <li class="moj-pagination__item moj-pagination__item--next">
      <a class="moj-pagination__link" href="#">Next</a>
    </li>
  </ul>
</nav>

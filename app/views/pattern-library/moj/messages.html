Messages

Use the MOJ messages component to show a conversation thread between caseworkers and applicants.

This prototype uses locally-styled variants of the MOJ messages component — moj-messages-container-local, moj-message-list-local and moj-message-item--sent-local — to accommodate layout differences in the casework interface. See the custom patterns section for more detail on these variants.

Standard message thread

Example
  • Sarah Watson

    Hello, I wanted to check whether my application has been received and when I can expect a decision.

    10 Sep 2025, 09:14am
  • Jane Smith, Case officer

    Thank you for getting in touch. Your application has been received and is currently under review. We aim to complete the review within 20 working days.

    10 Sep 2025, 11:02am

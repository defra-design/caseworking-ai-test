Filter

Use the MOJ filter component to help users narrow down a list of results by one or more criteria.

In this prototype the filter panel appears on the case list to let caseworkers filter by status, assigned officer and scheme year. A radio-button variant of this component is used on the calculations pages — see the Calculations filter custom pattern.

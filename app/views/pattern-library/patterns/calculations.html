Calculations Experimental

How a caseworker reviews a grant or payment calculation, finds a land parcel, and sees what has changed since the previous calculation run.

Scaffold — consolidate from the existing custom patterns (calculations filter, changed-value highlight, check-result tags) into one page-level pattern.

Anatomy

  1. Parcel search — accessible-autocomplete + Find button on one row (.parcel-search__row).
  2. Calculations filter — the custom filter variant.
  3. Calculation table — with changed-value highlight on cells that differ from the last run.
  4. Check results — Pass / Fail / Action tags per check.

Changed-value highlight
